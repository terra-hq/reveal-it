Simple Reveal Animation

This animation can be applied to any element with the class '.js--reveal-it'. The animation will start as soon as the element is initialized. You can control the animation by calling play and pause methods.

// Import the RevealIt class
import RevealIt from "./RevealIt";

// Find the first element with the class '.js--reveal-it'
const firstRevealElement = document.querySelector(".js--reveal-it");

// Initialize the animation on the element
var simpleReveal = new RevealIt({ element: firstRevealElement });

// Control the animation
simpleReveal.getAnimation().play(); // Start the animation
simpleReveal.getAnimation().pause(); // Pause the animation

The RevealIt class applies a GSAP animation to the specified element. You can control the animation by using the getAnimation method to access the GSAP timeline and then calling play or pause.

Custom Reveal Animation

This example demonstrates creating a custom animation with specific parameters. You can specify options such as:

  • duration: The length of time (in seconds) the animation runs.
  • opacity: The target opacity for the animated element, ranging from 0 (fully transparent) to 1 (fully opaque).
  • yoyo: A boolean value that, when set to true, causes the animation to go back and forth, effectively repeating the animation in reverse.
  • repeat: The number of times the animation should repeat. -1 indicates an infinite loop.
  • ease: Defines the rate of change of the animation. Examples include linear, power1.inOut, elastic.out, etc. GSAP offers a wide range of easing functions.
  • delay: Time in seconds before the animation starts after being triggered.
  • stagger: Used for animating multiple elements with a small delay between the start of each element's animation. Useful for sequenced animations.
  • scale, x, y, rotation: Transform properties for scaling, moving (along the x and y axes), and rotating the element.

Custom reveal animation with specific parameters...

                    
                    
// Import the RevealIt class
import RevealIt from "./RevealIt";

// Create a custom reveal animation
const customReveal = new RevealIt({
    element: document.querySelector(".js--reveal-it-custom"),
    options: {
        duration: 2,
        opacity: 0,
        yoyo: false,
        repeat: 3
    }
});

The custom RevealIt instance allows you to define specific animation properties. This provides flexibility to tailor the animation to fit different scenarios or UI elements.

Callbacks in RevealIt Animations

This example demonstrates how to use callbacks such as onStart and onComplete in RevealIt animations. These callbacks allow you to execute custom code at the beginning and end of the animation.

Element with RevealIt animation and callbacks...



// Import the RevealIt class
import RevealIt from "./RevealIt";

// Create a RevealIt instance with callbacks
const revealWithCallbacks = new RevealIt({
    element: document.querySelector(".js--reveal-it-callback"),
    options: {
        // ... other animation options ...
    },
    onStart: () => console.log("Animation started"),
    onComplete: () => console.log("Animation completed")
});

The callbacks onStart and onComplete are useful for adding additional logic or effects at the start and end of the animation, respectively. They can be defined when creating a new instance of RevealIt.
