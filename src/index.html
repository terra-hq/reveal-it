  • element (HTMLElement): The DOM element to which the reveal animation will be applied.
  • isActive (boolean): Returns true if the animation is currently playing.
  • play (function): Method to start the reveal animation.
  • pause (function): Method to pause the reveal animation.
  • getAnimation (function): Returns the GSAP timeline instance for more advanced control.
  • duration (number): Duration of the animation in seconds.
  • opacity (number): Target opacity for the animation.
  • yoyo (boolean) = true: Enables yoyo effect, making the animation repeat in reverse.
  • repeat (number) = -1: Defines the number of times the animation should repeat. -1 for infinite.

Simple Reveal Animation

This animation can be applied to any element with the class '.js--reveal-it'. The animation will start as soon as the element is initialized. You can control the animation by calling play and pause methods.

                        // Import the RevealIt class
                        import RevealIt from "./RevealIt";
        
                        // Find the first element with the class '.js--reveal-it'
                        const firstRevealElement = document.querySelector(".js--reveal-it");
        
                        // Initialize the animation on the element
                        var simpleReveal = new RevealIt({ element: firstRevealElement });
        
                        // Control the animation
                        simpleReveal.getAnimation().play(); // Start the animation
                        simpleReveal.getAnimation().pause(); // Pause the animation

The RevealIt class applies a GSAP animation to the specified element. You can control the animation by using the getAnimation method to access the GSAP timeline and then calling play or pause.

Multiple Reveal Animations

This example demonstrates controlling multiple animations at once. Each element with the class '.js--reveal-it--multiple' will have its own animation instance.

                    // For multiple elements
                    var reveals = [];
                    document.querySelectorAll('.js--reveal-it--multiple').forEach(function(element) {
                        reveals.push(new RevealIt({ element: element }));
                    });
                    document.getElementById('toggleAllAnimations').addEventListener('click', function() {
                        reveals.forEach(function(reveal) {
                            if (reveal.getAnimation().isActive()) {
                                reveal.getAnimation().pause();
                            } else {
                                reveal.getAnimation().play();
                            }
                        });
                    });

For multiple elements, each '.js--reveal-it--multiple' class element is given its own RevealIt instance. The 'toggleAllAnimations' button will control the animations of all these instances simultaneously.
